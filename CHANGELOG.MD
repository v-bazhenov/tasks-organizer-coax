# CHANGELOG

The file contains important changes from version to version. Started from 14.03.2021


## v1.0.2 - 21.04.21

### Features

* Sending email when user making sign up
* Reset password
* Email reminder for a todo task
* Notifications when user makes actions
* Fixtures
* Logging
* Unit Tests
* Caching
* Pagination
* Ability to attach image to todo task
* Ability to remind user about todo task by setting remind time using a calendar and timer
* Custom actions for admin in admin panel to work with multiple todo tasks
* Fields validation on model level
* Profile page where the user can add/update his/her personal information
* Django-debug-toolbar in DEBUG mode
* Django-extensions-package generated file with all possible urls
* Dockerized project with Postgresql
* Type annotations
